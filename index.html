Excel Upload

Upload an Excel file to import multiple refund records at once. The file should contain columns for Branch, Order Date, Refund Date, Basket Amount, Refund Amount, Reason, Status, and Notes.

1 Upload Excel File

Upload your Excel file using the upload area below. The system will automatically read and import the data.

2 Review Data

Preview your data in the table below. The system will automatically generate Order IDs for each record.

3 Import Data

Select the rows you want to import and click "Import Selected" to add them to the system.
